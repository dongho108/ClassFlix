## About ClassFlix
Classflix는 여러 사이트에 퍼져있는 인터넷 강의들을 모아놓은 플랫폼입니다. <br><br>
나에게 맞는 강의를 찾으려면 여러번 검색해서 정리해야했지만 이제는 그럴 필요 없습니다. <br><br>
한 사이트 안에서 듣고싶은 강의를 찾고, 별점을 확인하고, 리뷰를 확인하세요. <br><br>
